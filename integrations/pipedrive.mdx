## Pipedrive API Token

You will need a Pipedrive API token to integrate with ListingOpen. To create one, follow these simple steps:

1. Visit your Pipedrive Dashboard.
2. Click your profile in the top right corner of the page, then click "Personal Preferences" under "My Account".
<img src="https://listingopen-photos.s3.us-east-2.amazonaws.com/guides/pipedrive_1.gif" />
3. In the "Personal Preferences" page, navigate to the "API" tab.
4. Copy your personal API token or, if necessary, generate a new token.
<img src="https://listingopen-photos.s3.us-east-2.amazonaws.com/guides/pipedrive_2.gif" />
5. You can [save your API token to ListingOpen](/account/credentials) to avoid repeating this process.

## Add Integration

Once you have your Pipedrive API token, you are ready to integrate with ListingOpen.

1. Under your desired listing, navigate to the "Integrations" tab.
2. Click Pipedrive.
3. Paste your Pipedrive API token into the "API Token" field.
4. Click "Save & Publish" to finish adding the integration.
5. You'll know the integration is working when it is labelled "Active".

## Actions

Every new visitor to the listing will automatically result in:
* Contact created in Pipedrive
* Lead created for the Contact
* Note with sign-in answers attached to the Contact
